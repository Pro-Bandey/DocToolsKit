What is a URL Slug?

  • A slug is the part of a URL that identifies a page in a human-readable format
  • Example: example.com/blog/my-awesome-post
  • Best practices: Use hyphens, keep it short, include keywords, avoid special characters
  • SEO-friendly slugs improve search rankings and user experience
